# Example Reports

This directory contains example reports generated by the Technical Report Generator to demonstrate its capabilities.

## Available Examples

### example_report (Academic Report)

Generated from a machine learning practical notebook analyzing credit card fraud detection using various classification algorithms.

**Available Formats:**
- **Markdown** (`example_report.markdown`) - Lightweight, readable format with full content
- **DOCX** (`example_report.docx`) - Microsoft Word format for easy editing
- **PDF** (`example_report.pdf`) - Professional, print-ready format

**Report Highlights:**
- Comprehensive analysis of credit card fraud detection
- Comparison of multiple classification algorithms (SVC, Logistic Regression, Decision Tree)
- Automated diagram generation (Mermaid)
- Performance metrics and evaluation tables
- Proper citations in IEEE format
- Structured sections: Abstract, Introduction, Methodology, Results, Discussion, Conclusion

## How to Generate Your Own Reports

```bash
# Basic usage
python src/main.py --input your_notebook.ipynb --type academic --format all

# Specify output location
python src/main.py --input your_notebook.ipynb \
  --type academic \
  --format all \
  --output examples/output/my_report

# With additional options
python src/main.py --input your_notebook.ipynb \
  --type academic \
  --format docx \
  --title "My Custom Title" \
  --institution "Your Institution" \
  --include-code-snippets
```

## Report Types

- **academic** - Lab reports and semester projects
- **internship** - Internship documentation and work reports
- **industry** - Technical documentation for industry projects
- **research** - Research summaries and papers

## Output Formats

- **markdown** - Plain text with Markdown formatting
- **docx** - Microsoft Word document
- **pdf** - PDF with professional formatting
- **all** - Generate all formats simultaneously

## Viewing the Examples

- **Markdown**: Open `.markdown` files in any text editor or Markdown viewer
- **DOCX**: Open with Microsoft Word, Google Docs, or LibreOffice Writer
- **PDF**: Open with any PDF reader
